# Ledger AI CRM 🌌

An AI-powered CRM platform tailored for managing high-value architectural contracts, lead intelligence, sales pipelines, and document analytics. Designed with a premium "Digital Astral" UI philosophy, featuring deep dark mode aesthetics and intelligent glassmorphism accents.

## 🌟 Key Features

*   **Intelligence Dashboard**: A birds-eye view of your quarterly growth metrics and AI-flagged urgent client signals.
*   **Kanban Sales Pipeline**: Orchestrate high-value contracts through customizable pipeline stages from Discovery to Won. 
*   **Smart Lead Management**: View auto-generated lead scoring with embedded sentiment analysis from the system's "AI Intelligence Report."
*   **Contacts Directory**: Full searchable rolodex of your highest priority clients, partners, and prospects.
*   **Predictive Reports & AI Hub**: Automatically generated market reports, revenue forecasting, and team efficiency tracking.
*   **Interactive Task & Team Calendar**: Track every client touchpoint, team meetings, and urgent tasks.
*   **Document Intelligence**: Instantly view contract progress, e-signature tracking, and AI-flagged clause alerts.
*   **Automated Marketing Segmentations**: Track outbound sequences, open rates, and click tracking metrics.

## 🚀 Tech Stack

*   **Framework**: [Next.js](https://nextjs.org/) (App Router)
*   **Language**: [TypeScript](https://www.typescriptlang.org/)
*   **Styling**: Pure CSS / Variables (Vanilla CSS) for granular control over the design system.
*   **Icons**: [Lucide React](https://lucide.dev/)
*   **Charting Data**: [Recharts](https://recharts.org/)

## 🎨 Design System: "Digital Astral"
The platform strictly adheres to a "No-Line" rule—using nuanced surface colors instead of standard 1px borders to denote hierarchy across components:
*   `--surface`: `#060e20`
*   `--surface-container`: `#0f1930`
*   `--primary`: `#a3a6ff`
*   `--secondary`: `#34b5fa`

## 🛠️ Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the pages by modifying `src/app/page.tsx` or the styles inside `src/app/globals.css`.

## 📦 Deployment via Vercel

This repository is optimized for [Vercel](https://vercel.com/new). Pushing to the `main` branch will automatically trigger a new deployment. Ensure you use the default general Next.js presets if prompted during the Vercel import process.
